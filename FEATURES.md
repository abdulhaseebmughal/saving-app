# SaveIt.AI - Main Features

**Live App:** https://saving-app-eight.vercel.app/

## Core Features

### 1. **Smart Link Saving**
- AI-powered metadata extraction (title, description, images)
- Platform detection (YouTube, Instagram, GitHub, Twitter, etc.)
- Auto-categorization (Education, Tech, AI, Programming, etc.)
- Beautiful link cards with platform badges

### 2. **Sticky Notes Board**
- Infinite drag-and-drop canvas
- 8 color palettes for notes
- Pin/unpin functionality
- Real-time positioning and z-index management
- Multiple font sizes (S/M/L)
- Minimize/maximize notes
- Search and filter

### 3. **Diary Notes**
- Digital sticky notes with lined paper design
- Color customization (Yellow, Pink, Blue, Green, Purple, Orange)
- Pin important notes to top
- Rich text editing
- Auto-save functionality

### 4. **File Manager**
- Upload any file type (up to 10MB)
- Files stored as base64 in MongoDB
- Download exact uploaded files
- Industry/category organization
- Auto-detect file categories (PDF, Document, Image, Code, etc.)
- Drag-and-drop upload

### 5. **Project Management**
- Create and organize projects
- Industry categorization
- Color-coded projects
- File count tracking

### 6. **Organization System**
- Custom industries/categories
- Icon and color customization
- Item count tracking
- Nested organization

### 7. **Authentication**
- OTP-based email verification
- Secure JWT tokens
- Password recovery
- User profiles

### 8. **Admin Portal**
- Access via Ctrl+Shift+A
- Real-time statistics dashboard
- User data management
- System monitoring

### 9. **PWA (Progressive Web App)**
- Install as native app
- Offline support
- Home screen installation
- Service worker caching

### 10. **AI Integration**
- Google Gemini API for smart categorization
- Automatic metadata extraction
- Intelligent link analysis

## Tech Stack

**Frontend:** Next.js 16, React 19, TypeScript, Tailwind CSS, Framer Motion, React-DND
**Backend:** Node.js, Express, MongoDB, Mongoose
**AI:** Google Gemini API
**Deployment:** Vercel (Frontend & Backend)
